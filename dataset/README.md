# Amazon-Product-Sales-Analysis
A data analytics project analyzing amazon products sales through tools including Excel, MySQL and PowerBI in order to obtain meaningful insights.
## Project Overview

This project focuses on analyzing an Amazon product dataset to uncover meaningful insights into product pricing, discounts, customer ratings, and category performance.

The project follows a complete data analytics workflow, starting with data cleaning in Microsoft Excel, followed by data exploration and business analysis using MySQL, and finally presenting the findings through an interactive Power BI dashboard.

The objective was not only to practice SQL queries but also to transform raw data into meaningful business insights using data visualization.

##  Objectives

- Explore and understand the dataset.
- Analyze product pricing and discount patterns.
- Compare customer ratings across product categories.
- Identify the most expensive and most reviewed products.
- Build an interactive dashboard to communicate business insights.



##  Tools & Technologies

- **Microsoft Excel** – Data Cleaning
- **MySQL** – Data Analysis
- **Power BI** – Data Visualization & Dashboard

---

##  Dataset

**Source:** Amazon Product Dataset (Kaggle)

The dataset contains information about Amazon products, including:

- Product Categories
- Product Prices
- Discount Percentages
- Customer Ratings
- Rating Counts

---

##  Data Cleaning

The dataset was cleaned in Microsoft Excel before importing it into MySQL.

Cleaning steps included:

- Removing unnecessary columns
- Converting price columns into numeric format
- Formatting discount percentages
- Checking for missing values
- Preparing the dataset for SQL analysis


##  SQL Analysis

The SQL analysis was divided into three sections.

### Section 1: Data Exploration

- Previewed the dataset
- Counted unique products and categories
- Explored price and rating ranges
- Identified missing values

### Section 2: Product Performance Analysis

- Calculated average prices
- Identified top-priced products
- Compared average ratings by category
- Analyzed discount percentages
- Found products with the highest ratings and review counts

### Section 3: Business Insights

- Compared products against overall averages
- Identified highly rated products with large discounts
- Analyzed category-level performance
- Calculated price differences
- Explored discount trends

### SQL Concepts Used

- SELECT
- WHERE
- DISTINCT
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions
- Subqueries
- LIMIT

##  Power BI Dashboard

The interactive dashboard includes:

- Total Products
- Total Reviews
- Average Rating
- Average Selling Price
- Average Discount Percentage
- Discount by Category
- Top Reviewed Products
- Top Expensive Products
- Category Filter (Slicer)

---

## Key Insights

- Electronics-related categories contained the largest number of products.
- Most products maintained customer ratings above 4.0.
- Products with higher discounts did not necessarily receive lower ratings.
- A small number of products accounted for a significant share of total customer reviews.
- Discount percentages varied noticeably across categories.
